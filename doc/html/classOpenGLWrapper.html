Detailed Description

OpenGLWrapper gives a C++ like interface to easily build an OpenGL Window. It takes a lot of freedom away but it offers a very easy to use interface. It depends on OpenGL, SFML, GLFW and GLEW (as u can see in the includes).

int OpenGLWrapper::run ( )

Calls the OpenGL Runtime. This will open the OpenGL window (call of overloaded Constructor is absolutely necassary before running!) and open it endlessly.
